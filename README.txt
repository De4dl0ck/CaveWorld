CaveWorld 1.0

In dieser Mod beginnt man unter der Erde in einem kleinen Raum mit einem Eichenbaum, da die Oberfläche verstrahlt ist.
Man muss versuchen unter der Erde zu überleben. Die Oberfläche ist nur mit einem Strahlnenschutzanzug begehbar.

FEATURES:

- Welthöhe auf 128 Blöcke erhöht
- Weltgenerator so verändert, dass eine flache, lebensfeindliche Umgebung erzeugt wird
- Strahlenschutzanzug hinzugefügt
- Verstrahlungs-Effekt hinzugefügt
- Start in einem kleinen Raum mit einem Baum
- Anzahl der Dungeons drastisch erhöht
- Chest Loot beinhaltet Saplings anderer Baumsorten
- Mobs spawnen den ganzen Tag an der Oberfläche, da es keinen Tag-Nacht Zyklus mehr gibt
